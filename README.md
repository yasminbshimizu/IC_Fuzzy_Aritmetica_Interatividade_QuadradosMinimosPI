## 📝 Resumo

<p align="justify">
O método de quadrados mínimos é utilizado para o ajuste de curvas baseado em dados, permitindo realizar previsões de um fenômeno. A partir das curvas obtidas pelo ajuste, quando se deseja prever valores em x (entrada) desconhecidos a partir de valores em y (saída) conhecidos, é possível aplicar a inversão direta de problemas inversos para obter os valores desejados. Ao lidar com dados estatísticos, biológicos ou experimentais, é preciso considerar as imprecisões intrínsecas a tais dados. Nesse contexto, a Teoria de Conjuntos Fuzzy surge para construir modelos matemáticos mais realistas, considerando a incerteza associada ao fenômeno e modelada por números fuzzy. Se as incertezas envolvidas em diferentes números fuzzy afetam uma a outra, é necessário utilizar o conceito de interatividade. O presente projeto tem o objetivo de estudar problemas inversos descritos por quadrados mínimos, considerando incerteza e interatividade fuzzy, a partir de diferentes aritméticas fuzzy. Além disso, o projeto focará também na implementação computacional do método visando diferentes aplicações a dados experimentais para validar a metodologia e comparar o desempenho das variadas abordagens.

**Palavras-chave:** Teoria de Conjuntos Fuzzy • Interatividade Fuzzy • Aritmética Fuzzy • Quadrados Mínimos Fuzzy • Problemas Inversos

</p>
