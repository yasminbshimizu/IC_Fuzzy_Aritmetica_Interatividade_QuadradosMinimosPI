# IC_Fuzzy_Aritmetica_Interatividade_QuadradosMinimosPI
Este repositório contém diferentes abordagens para o método de quadrados mínimos fuzzy e seu problema inverso, explorando aritméticas fuzzy e interatividade, como projeto de Iniciação Científica na Ilum Escola de Ciência sob orientação do Prof° Dr. Vinícius F. Wasques.
